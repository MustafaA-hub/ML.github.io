`list()`: **splitting words**
````Python
word = "hello"
chars = list(word) # chars = ['h', 'e', 'l', 'l', 'o']
````

`.split()`: **splitting strings** 
````Python
s = "apple,banana,cherry"
fruits = s.split(",")  # fruits = ['apple', 'banana', 'cherry']
````

Dictionary
````Python
my_dict = {"apple": 1, "banana": 2, "cherry": 3}
value = my_dict["apple"] # value = 1
````
